# Flores Para a Minha Flor

Este projeto foi criado como uma homenagem romântica em forma de arte digital. A proposta é apresentar uma animação delicada de flores em uma página única, com atmosfera noturna, composição elegante e foco em uma experiência visual afetiva.

O projeto é desenvolvido apenas com HTML e CSS, sem uso de JavaScript ou bibliotecas externas. A ideia é manter o arquivo leve, fácil de abrir no navegador e simples de personalizar depois, alterando textos, cores e elementos visuais conforme necessário.

## Objetivo

Criar uma página visualmente bonita e simbólica para transmitir carinho de forma criativa, usando animações suaves, fundo escuro e uma estética romântica inspirada em flores.

## Tecnologias

- HTML5
- CSS3

## Características

- animação visual feita sem JavaScript
- estilo romântico com fundo noturno
- estrutura simples e fácil de editar
- projeto pensado para funcionar direto no navegador

## Personalização

O conteúdo do projeto pode ser adaptado facilmente para:

- mudar a frase principal
- trocar paleta de cores
- ajustar flores, folhas e composição visual
- preparar a página para publicação no GitHub Pages
